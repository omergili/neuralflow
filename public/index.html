Add it to your site

<script src="https://cdn.jsdelivr.net/npm/@neuralflow/ai-act/dist/badge.min.js"
  data-operator="Your Company"
  data-ai-system="Claude"
  data-lang="en">
</script>

That's it. The badge appears. Click it for the full disclosure. Machine-readable metadata is injected into your page's <head>.

GitHub Documentation

Why this exists

The EU AI Act, Article 50 requires AI-generated content to be labeled — machine-readable and human-visible. Enforcement begins August 2, 2026. Penalties: up to EUR 15 million or 3% of annual global turnover.

There was no standard badge for this. Now there is.

What it does

  • Visible "AI Transparent" badge (configurable position)
  • Disclosure popup on click (Article 50 compliant text)
  • JSON-LD structured data in <head>
  • HTML <meta> tags for AI disclosure
  • German and English support
  • Zero dependencies, zero cookies, zero tracking

For developers

npm install @neuralflow/ai-act
import { createDisclosure, generateMetadata } from '@neuralflow/ai-act';

const disclosure = createDisclosure({
  operator: 'Your Company',
  aiSystem: 'Claude',
  lang: 'de',
});

console.log(disclosure.html);
console.log(disclosure.jsonLd);

CLI

npx @neuralflow/ai-act generate --operator "Your Company" --ai-system "Claude"
npx @neuralflow/ai-act check https://your-site.com
