Vor </body> einfügen. Fertig. Das Badge erscheint in der Ecke. Klick darauf für die vollständige Offenlegung. Maschinenlesbare Metadaten (JSON-LD + Meta-Tags) werden automatisch in den <head> eingefügt.

Diese Seite nutzt das Badge. Schau unten rechts in die Ecke. Klick drauf.

Oder prüfe jede Website auf AI Act Konformität — kostenlos, sofort, ohne Anmeldung.

Was es macht

  • Sichtbares Badge — „KI Transparent" in konfigurierbarer Position (unten rechts/links, oben rechts/links)
  • Offenlegungs-Popup — Klick für den vollständigen Artikel-50-konformen Text (Deutsch + Englisch)
  • JSON-LD Metadaten — schema.org Structured Data im <head>
  • Meta-Tags — ai-generated, ai-system, ai-operator
  • Null Footprint — keine Abhängigkeiten, keine Cookies, kein Tracking, keine externen Requests

Funktioniert mit allem

WordPress, Shopify, Shopware, React, Next.js, Vue, Nuxt, Svelte, reines HTML — jede Website. Integrations-Anleitung.

Warum jetzt?

Als die DSGVO 2018 kam, wuchs der Cookie-Consent-Markt von null auf 2,3 Milliarden Dollar. OneTrust erreichte 500 Mio. USD ARR. Cookiebot bedient 2,3 Mio. Websites.

Der EU AI Act folgt dem gleichen Muster. Artikel 50 wird ab dem 2. August 2026 durchgesetzt. Jede Website mit KI-generierten Inhalten muss konform sein. Der Unterschied: Diesmal ist das Tool kostenlos und Open Source.
